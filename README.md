## 📌 Project Overview
This project demonstrates an end-to-end data analysis pipeline using MySQL and Excel. The goal was to simulate an e-commerce environment, perform data entry and manipulation using SQL, and generate business insights through an Excel dashboard.

---

## 🛠 Tools Used
- MySQL (Database creation, data entry, and querying)
- Microsoft Excel (Data analysis and dashboard creation using Pivot Tables)

---

## 📂 Project Structure
- `SQL Scripts/` → Database creation and data insertion scripts
- `Data/` → Exported dataset from MySQL
- `Dashboard/` → Excel dashboard screenshots

---

## ⚙️ Data Pipeline Process

### 1. Database Creation (MySQL)
- Created an e-commerce database with the following tables:
  - Customers
  - Products
  - Orders
  - Order Items
- Defined relationships using primary and foreign keys

### 2. Data Entry & Manipulation
- Inserted sample data into all tables
- Ensured referential integrity between tables
- Structured data for analysis

### 3. Data Export
- Exported cleaned data from MySQL into Excel

### 4. Data Analysis (Excel)
- Used Pivot Tables to analyze:
  - Total sales by product
  - Customer purchase behavior
  - Order trends by status
- Built a dashboard to visualize key metrics
